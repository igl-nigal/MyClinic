**DESCRIPTION:**
MyClinic is an app built for mobile to help users track their health status and suggest them ways to improve their lifestyle and to alert medical professionals if needed

MyClinic is a comprehensive platform that:

Tracks: Records key metrics like daily steps, water intake, and sleep duration.

Analyzes: Generates trend charts and provides an "Overall Health Score."

Recommends: Offers personalized lifestyle suggestions based on user data.

Alerts: Automatically notifies preselected medical professionals if abnormal health indicators are detected.
